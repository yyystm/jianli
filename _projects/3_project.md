该项目是当前阶段第二项较完整的研究型工程项目，对应二作 SCI 论文与国家级大创经历，担任第二负责人。

- **项目目标：** 构建双阻态耦合忆阻 Hopfield 神经网络模型，研究其复杂动力学行为，并完成模拟电路与 FPGA 实现验证。
- **理论与建模：** 围绕 DMHNN 的 Lyapunov 指数、分岔行为和混沌动力学开展分析。
- **主要工作：** 负责 MHNN 模型的模拟电路推导与 Multisim 仿真验证，完成忆阻突触串扰模块、双曲正切函数模块及主电路等效设计。
- **验证方式：** 通过 x-y、y-z 相平面输出波形仿真，验证模拟电路输出与数值仿真的一致性，并配合完成 FPGA 实现与应用展示。
- **阶段成果：** 第二作者 SCI 论文 *Dynamic Analysis, FPGA Implementation and Application of Memristive Hopfield Neural Network with Synapse Crosstalk* 发表于 *Electronics*。
